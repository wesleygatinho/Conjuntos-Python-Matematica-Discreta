# Atividade Prática - Teoria dos Conjuntos com Python

Este projeto foi desenvolvido como parte da disciplina de **Lógica e Matemática Discreta** (Engenharia da Computação - UFMA). O objetivo é aplicar conceitos fundamentais da teoria dos conjuntos através da programação.

## Descrição

O programa realiza operações matemáticas entre dois conjuntos de números inteiros:

1. **Conjunto A:** Definido pelo usuário (entre 4 e 8 elementos).
2. **Conjunto B:** Gerado aleatoriamente pelo sistema (entre 4 e 8 elementos).

O sistema calcula e exibe a união, interseção, diferença e a cardinalidade (tamanho) dos conjuntos resultantes.

## Como executar

Certifique-se de ter o **Python 3** instalado em sua máquina.

1. Clone este repositório ou baixe o arquivo `main.py`.
2. Abra o terminal na pasta do arquivo.
3. Execute o comando:

   ```bash
   python main.py
